แบบฝึกหัด
  1. ในบรรทัดที่มีเลข 1 2 และ 3 จะได้ค่าออกมาเป็นอะไร
    true
    null
    undefined
  2. ใช้ __proto__ ในการกำหนด prototype object ดังนี้
    pockets → bed → table → head. 
                    
let head = {
    glasses: 1
};
let table = {
    pen: 3,
    __proto__: head
};
let bed = {
    sheet: 1,
    pillow: 2,
    __proto__: table
};
let pockets = {
    money: 2000,
    __proto__: bed
};
  3. การเขียนโค๊ดแบบนี้ object ไหนจะเป็นตัวที่ได้ property full ไป
    rabbit
  4. การเขียนโค๊ดแบบนี้ กระเพาะจะถูกแชร์กันจะแก้ไขยังไงดี
                    
let hamster = {

    eat(food) {

        if(!this.stomach) this.stomach = [];

        this.stomach.push(food);
    }
};
