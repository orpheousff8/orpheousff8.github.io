  4. ให้เขียน loop ทั้งแสดงเลข 2 ถึง 10 ออกมา 
                    
let txt;
for (let i = 2; i < 11; i++) {
    txt += i + ` `;
}
alert(txt);
  5. เปลี่ยน code for loop ด้านล่างนี้ให้เป็น while loop โดยที่ผลลัพธ์ยังเหมือนเดิม

                    
let i = 0;
while (i < 3)
{
    alert( `number ${i}!` );
    i++;
}
  6. ให้เขียนเกมส์ทายตัวเลขสำหรับเล่นสองคน โดย
    • ให้ผู้เล่นคนแรกพิมพ์เลขใส่ใน prompt ที่อยู่ระหว่าง 1 ถึง 100 โดยไม่ให้ผู้เล่นคนที่สองรู้ว่าตัวเลขเป็นอะไร
    • และให้ผู้เล่นคนที่สองทายเลขโดยการพิมพ์เลขใส่ใน prompt จนกว่าจะถูก ถ้าไม่ถูก จะต้องบอก ด้วยว่าเลขที่ผู้เล่นคนที่สองพิมพ์เข้ามา มากกว่า หรือ น้อยกว่าคำตอบนั้น
