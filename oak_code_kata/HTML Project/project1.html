ประมูล.คอม

เว็บไซต์ประมูลสินค้าออนไลน์ในไทยรายแรก

    ประมูลดอตคอม (อังกฤษ: Pramool.com) เป็นเว็บไซต์ประมูลสินค้าออนไลน์ในไทยรายแรก ก่อตั้งเมื่อปี พ.ศ. 2542 โดยนายนาถพงศ์ อัมพรอร่ามเวทย์ (ชื่อเล่น: หยง) ซึ่งเสียชีวิตในเดือนสิงหาคม พ.ศ. 2558 จากโรคลิ้­นหัวใจรั่วเฉียบพลัน ภายในเว็บไซต์มีเว็บบอร์ดพูดคุยแลกเปลี่ยนโดยแบ่งตามหมวดหมู่ ประมูลดอตคอมมีชื่อเสียงในทางที่ไม่ดีมากนัก เนื่องจากถูกมองว่าเป็นเว็บไซต์ที่มีผู้ใช้จำนวนมากโพสต์ข้อความหรือกระทู้ที่มีถ้อยคำหยาบคาย และสนทนากันเรื่องเกม, การ์ตูน หรือเรื่องไม่มีสาระต่าง ๆ อีกทั้งเป็นที่รู้จักจากภาษาวิบัติที่แพร่หลาย ประมูลดอตคอมโด่งดังในราวปี 2546 - 2552 และหมดความนิยมลงไปในปัจจุบัน ประมูลดอตคอมมักถูกเปรียบเทียบเป็นแหล่งซ่องสุมด้านมืดเพราะมักหยิบยกและพูดคุยเรื่องราวที่ค่อนข้างเรื่องมืด ๆ เช่นการแจกภาพโป๊เปลือยอนาจาร หรือภาพสยองขวัญที่เมื่อผู้ชมคลิกดูแล้วจะตกใจ หรือการโปรโมทระบบเซิร์ฟเวอร์เกมเถื่อน
