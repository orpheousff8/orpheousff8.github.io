1 1.1 CSS syntax ประกอบด้วยอะไรบ้าง
  1. Selector คือ ส่วน HTML tag ที่เราต้องการจะให้แสดงผล เช่น <table> ก็จะแสดงผลในส่วนของตาราง

1.2 การเขียน CSS มีกี่แบบอะไรบ้าง 3 แบบ
  1. Inline คือการเขียน CSS แทรกลงไปในบรรทัดนั้นๆ ของ tag HTML เลย
  2. Internal คือการเขียนแยก CSS ออกจาก tag HTML แต่อยู่ในหน้าเดียวกัน โดยแทรกเอาไว้ที่ <head></head>
  3. External คือการเขียนแยก CSS ออกมาเป็นไฟล์ .css ต่างหาก แล้วนำเข้าไปใช้โดยแทรก

1.3 selector มีกี่แบบอะไรบ้าง
    12 แบบ
  1. X เลือก tag <X> ทุกตัว
  2. * เลือก ทุกๆ element
  3. #X เลือก โดยใช้ id
  4. .X เลือก โดยใช้ class
  5. X[abc] เลือก <X> ทุกตัว ที่มี attribute abc
  6. X[abc="xyz"] เลือก <X> ทุกตัว ที่มี attribute abc ที่มีค่าเป็น xyz เป๊ ๆ
  7. X[abc*="xyz"] เลือก <X> ทุกตัว ที่มี attribute abc ที่มีค่า xyz อยู่ในนั้น
  8. X[abc^="xyz"] เลือก <X> ทุกตัว ที่มี attribute abc ที่มีค่าขึ้นต้นด้วย xyz
  9. X[abc$="xyz"] เลือก <X> ทุกตัว ที่มี attribute abc ที่มีค่าลงท้ายด้วย xyz
  10. X[abc~="xyz"] เลือก <X> ทุกตัว ที่มี attribute abc ที่มีค่า xyz ที่มีช่องว่างนำหน้า หรือตามหลัง
  11. Pseudo Classes เช่น X:first-child ใช้เลือก <X> ทุกตัว ที่เป็น child ตัวแรกของ parent ใด ๆ
  12. Pseudo Elements เช่น X::before ใช้แทรกคอนเทนต์หน้า <X> ทุกตัว
รายชื่อและตัวอย่าง Pseudo Classes, Pseudo Elements ทั้งหมด
1.4 combinator มีกี่แบบอะไรบ้าง 6 แบบ
  1. X, Y select ทั้ง X และ Y ทุกตัว
  2. X + Y เลือก เฉพาะ Y ที่อยู่ระดับเดียวกัน และชิดถัดจาก X เท่านั้น
  3. X ~ Y เลือก Y ที่อยู่ระดับเดียวกัน และถัดจาก X ทุกตัว
  4. X Y เลือก Y ที่เป็น descendant ของ X ทุกตัว
  5. X > Y เลือก เฉพาะ Y ที่เป็น child ของ X ลงไป 1 ขั้น เท่านั้น
  6. [abc][def] เลือก HTML tag ใด ๆ ที่มี attribute ทั้ง abc และ xyz

4 4.1 value ใน display property มีอะไรบ้าง
  1. inline - แสดง element เหมือน <span> โดยอยู่บรรทัดเดิม ส่วน property height กับ width จะไม่มีผล
  2. block - แสดง element เหมือน <p> โดยจะเริ่มบนบรรทัดใหม่ และใช้ความกว้าง container จนเต็ม
  3. contents - ทำให้ container หายไป
  4. flex - แสดง element แบบ flex โดยตัว container จะเป็นแบบ block
  5. grid - แสดง element แบบ grid โดยตัว container จะเป็นแบบ block
  6. inline-block - แสดง element อยู่ใน block container โดยปรับแต่ง property height กับ width ได้ แต่จะอยู่ในบรรทัดเดิมแบบ inline
  7. inline-flex - แสดง element แบบ flex โดยตัว container จะเป็นแบบ inline
  8. inline-grid - แสดง element แบบ grid โดยตัว container จะเป็นแบบ inline
  9. inline-table - แสดง element แบบ table โดยตัว container จะเป็นแบบ inline
  10. list-item - ให้ element มีพฤติกรรมเหมือน <li>
  11. run-in - แสดง element แบบ block หรือ inline ขึ้นกับเนื้อหา
  12. table - ให้ element มีพฤติกรรมเหมือน <table>
  13. table-caption - ให้ element มีพฤติกรรมเหมือน <caption>
  14. table-column-group - ให้ element มีพฤติกรรมเหมือน <colgroup>
  15. table-header-group - ให้ element มีพฤติกรรมเหมือน <thead>
  16. table-footer-group - ให้ element มีพฤติกรรมเหมือน <tfoot>
  17. table-row-group - ให้ element มีพฤติกรรมเหมือน <tbody>
  18. table-cell - ให้ element มีพฤติกรรมเหมือน <td>
  19. table-column - ให้ element มีพฤติกรรมเหมือน <col>
  20. table-row - ให้ element มีพฤติกรรมเหมือน <tr>
  21. none - ลบ element ออกไป
  22. initial - กำหนด property เป็นค่า default
  23. inherit - ให้ใช้ property ของ parent element
4.2 grid และ flex แตกต่างกันอย่างไร Flex ใช้เพื่อจัดการ layout 1 มิติ ส่วน Grid ใช้เพื่อจัดการ layout 2 มิติ
Flex เราสามารถจัดวางของต่างๆภายใน layout 1 แถว อาจจะเป็นแนวตั้งหรือแนวนอน
Grid เราสามารถจัดของไว้ตรงไหนก็ได้ใน grid layout เช่นให้ layout ของ header เต็ม row แรก 10 columns
บรรทัดต่อมาให้ menu เป็น 2 columns content เป็น 8 columns
4.3 justify and align แตกต่างกันอย่างไร justify-content เป็นการจัดตำแหน่งของ Flex Item ภายใน Flex Container
  • flex-start: จัด Flex Item ชิดตำแหน่งต้น*
  • flex-end: จัด Flex Item ชิดตำแหน่งท้าย*
  • center: จัด Flex Item อยู่ตรงกลาง
  • space-between: จัด Flex Item ให้มี margin ระหว่างแต่ละ item เท่า ๆ กัน
  • space-around: จัด Flex Item ให้มี margin ระหว่าง container หรือ item เท่า ๆ กัน
    • *ตำแหน่ง ต้น/ท้าย ขึ้นอยู่กับ flex-direction โดย
    • row: เรียงเป็นแถวแนวนอน จากซ้ายไปขวา
    • row-reverse: เรียงเป็นแถวแนวนอน จากขวาไปซ้าย
    • column: เรียงเป็นแถวแนวตั้ง จากบนลงล่าง
    • column-reverse: เรียงเป็นแถวแนวตั้ง จากล่างขึ้นบน
align-items เป็นการจัดตำแหน่งของ Flex Item ในแกนตั้งฉากกับ flex-direction ภายใน Flex Container
  • flex-start: จัด Flex Item ให้อยู่บนสุดของแกน
  • center: จัด Flex Item ให้อยู่ตรงกลางของแกน
  • flex-end: จัด Flex Item ให้อยู่ล่างสุดของแกน
5 5.1 position คืออะไร และ อะไรบ้าง เป็นการกำหนดตำแหน่งของ element เมื่อเทียบกับ container
  • top, left, bottom, right เป็นการกำหนดระยะห่างจากขอบของ container กำหนดเป็นค่าติดลบได้
    • นิยมกำหนดเป็นคู่ top/left, top/right, bottom/left, bottom/right อย่างใดอย่างหนึ่ง
    • ต้องใช้คู่กับ property position ที่ไม่ใช่ static จึงจะได้ผลลัพท์ตามต้องการ
  • position เป็นลักษณะการเทียบตำแหน่งกับ container
    • static (ค่าเริ่มต้น) จัดวางตำแหน่งตามที่กำหนดไว้ใน HTML
    • absolute เทียบตำแหน่งกับ container โดยตรง ถ้า container ไม่ได้กำหนดตำแหน่งแบบ static
      • ถ้า container เป็น body จะเทียบตำแหน่งกับ body
      • ถ้า container ไม่ใช่ body container ต้องมี position แบบ absolute, relative หรือ sticky
        จึงจะเทียบตำแหน่งกับ container มิฉะนั้นการกำหนด top, left, bottom, right จะไม่มีผลใด ๆ
      • ถ้ากำหนดตำแหน่งหลาย element พร้อมกันด้วยค่า top/left, bottom/right เดียวกัน มันจะทับกัน
    • fixed ตรึง element อยู่กับที่ ไม่เลื่อนตามสกรอลล์บาร์ และไม่ขึ้นกับ container
      • ถ้ากำหนดตำแหน่งหลาย element พร้อมกันด้วยค่า top/left, bottom/right เดียวกัน มันจะทับกัน
    • sticky ตรึง element อยู่กับที่ ไม่เลื่อนตามสกรอลล์บาร์ แต่จะเลื่อนลงมาสุดแค่ container ต้องกำหนดค่า top ด้วย (เป็น 0 ได้)
      • ถ้ากำหนดตำแหน่งหลาย element พร้อมกันด้วยค่า top/left, bottom/right เดียวกัน มันจะทับกัน
    • relative เทียบกับตำแหน่งเดิมที่มันควรจะอยู่ จากโค้ด HTML แล้วเลื่อนไปตามระยะที่กำหนด
      • ปกติจะเทียบระยะด้วย top/left เป็นหลัก ถ้าจะใช้ bottom/right ให้ใส่ระยะเป็นลบ
        (bottom: -10px; right: -20px; มีค่าเท่ากับ top: 10px; left: 20px;)
      • ถ้ากำหนดตำแหน่งหลาย element พร้อมกันด้วยค่า top/left, bottom/right เดียวกัน มันจะไม่ทับกัน
        เนื่องจากอ้างอิงตำแหน่งระหว่างกันด้วยโค้ด HTML
5.2 z-index คืออะไรใช้อย่างไร z-index คือลำดับของ element ในแกน z ของหน้าจอ (x แนวนอน, y แนวตั้ง)
เพราะฉะนั้นแกน z คือแนวลึกเข้าไปในจอ และออกนอกจอ
ค่าตั้งต้นของแต่ละ element คือ 0
ถ้า element มีค่า z-index เท่ากัน element ที่มาทีหลัง จะทับ element ที่มาก่อน
element ที่มีค่า z-index น้อยกว่า (เป็นจำนวนเต็มลบได้) จะอยู่ layer ล่างกว่า element ที่มีค่ามากกว่า
